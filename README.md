# Черепашка (Maximum path sum)

Задача о поиске пути максимальной суммы в таблице N×M. Движение только вправо или вниз.

## Формат ввода
```
N M
матрица N x M из чисел 0-100
```

## Формат вывода
- Первая строка: максимальная сумма
- Вторая строка: маршрут из D (вниз) и R (вправо)

## Решение
Динамическое программирование: dp[i][j] = grid[i][j] + max(dp[i-1][j], dp[i][j-1])

## Запуск
```bash
python main.py
```

## Пример
Вход:
```
3 3
1 2 3
4 5 6
7 8 9
```
Вывод:
```
29
DDRR
```
